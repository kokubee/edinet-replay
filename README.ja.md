# EDINET Replay（日本語）

> 正本は英語版 [README.md](README.md) です。本ファイルは補助的な日本語訳です。

日本の EDINET（金融庁の電子開示システム）提出書類を、**原典への追跡可能性を維持したまま**取得・展開・構造化する再現可能な抽出ツールキットです。

EDINET から公式の提出書類パッケージを取得し、生の原本を保全し、XBRL / Inline XBRL を [Arelle](https://arelle.org/) で解決して、原典へのリンクを保持した機械可読な忠実表現を出力します。

想定利用者は、データエンジニア・クオンツ研究者・XBRL実装者・ESGデータ利用者・学術研究者・データベンダー・機関投資家のデータ担当者です。一般個人投資家が直接使うことは想定していません。

> **現状: 初期段階。** 抽出器の実装より先に、出力の「契約」をバージョン付き JSON Schema（[`schemas/`](schemas/)）として定義しています。

## このプロジェクトの目的

EDINET は公開情報ですが、再現可能な利用は難しいままです（XBRL/iXBRL 構造の複雑さ、タクソノミの年度変化、抽出時の context/unit/dimension/来歴の欠落、正規化DBの変換根拠の不透明さ）。

本プロジェクトは**財務的な解釈ではなく忠実な再現**に集中します。原典の事実を第一に保全し、名寄せ・正規化・比較可能性判定・投資分析は別の下流レイヤーに置きます。

## やらないこと

- 異なるタグを自動で同一概念へ統合しない
- データの経済的意味を保証しない
- 投資助言をしない
- EDINET / 金融庁の公式製品ではない

> 本プロジェクトは金融庁と提携・承認関係にありません。
